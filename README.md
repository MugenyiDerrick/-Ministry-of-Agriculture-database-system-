# - Project Overview -
The Ministry of Agriculture is implementing a database system to sensitize coffee growers on
best practices for growing quality coffee. The ministry requires a system to maintain records of
all coffee farmers, track their production, plan and distribute coffee seedlings and other
agricultural resources, and facilitate access to extension workers for farm advisory services.
The system was developed using MySQL 8.0 as the database engine and VS Code as the
development environment. It covers farmer registration, farm management, production
tracking, resource inventory and distribution, extension worker advisory sessions, and role
based access control
